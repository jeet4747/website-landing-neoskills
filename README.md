## ğŸ”§ Customization

### Change Colors
Edit `tailwind.config.js`:
```javascript
colors: {
  neo: {
    dark: '#0a0f1f',        // Primary background
    darker: '#070b16',      // Darker variant
    green: '#00ff9c',       // Change accent green
    blue: '#00b8ff',        // Change accent blue
    gray: '#1a1f2e',        // Secondary background
  }
}
```

### Update Course Information
Edit `src/components/CoursesSection.jsx` - modify the `courses` array.

### Update Testimonials
Edit `src/components/TestimonialsSection.jsx` - modify the `testimonials` array.

### Update Contact Information
Edit `src/components/Footer.jsx` and `src/components/ContactForm.jsx` with actual details.
